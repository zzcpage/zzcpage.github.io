
#### 判断一个数是否是质数

```java
public boolean isPrime(int x){
        int  n = (int)Math.sqrt(x)+1 ;
        for(int i = 2 ; i <= n  ; i++){
            if(x % i == 0)
                return false ;
        }
        return true ; 
    } 
```

#### 奇数和偶数的性质

奇数通项公式可以写成$S_n=1+2d$。
偶数通项公式可以写成$S_n=2d$。

由奇数通项公式和偶数通项公式可以看出，奇数和偶数之间的差距就少了1，也就是说偶数+1等于奇数。所以一个区间内的奇数和偶数个数要么是一半一半，要么是一方比另一方多1。
